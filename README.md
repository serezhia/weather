## Features

 - 🖤 Black/White themes
 - 🌤️ OpenWeatherApi

## Prepair
```
1. flutter pub get
2. flutter gen-l10n
3. flutter pub run build_runner build --delete-conflicting-outputs  
4. flutter run --target  lib/main/main_production.dart || lib/main/main_development.dart 
```
